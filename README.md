# Secure Login System

## Overview
A secure login web application built using Flask and SQLite.

## Features
- User Registration
- User Login
- Password Hashing using Bcrypt
- SQL Injection Protection using Parameterized Queries
- Session Management
- Logout Functionality
- Input Validation

## Technologies Used
- Python
- Flask
- SQLite
- Flask-Bcrypt
